# Chat App API

A real-time chat application API built with Django REST Framework and Node.js for handling messaging functionality.

## Overview

A chat application backend providing user authentication and real-time messaging capabilities, built with a Django REST + Node.js hybrid architecture.

## Features

- **User Authentication** — Register and login
- **Real-time Messaging** — Send and receive chat messages
- **REST API** — Clean API endpoints for chat operations
- **Hybrid Backend** — Django REST + Node.js

## Tech Stack

- **Python** — Django REST Framework
- **Node.js** — Real-time messaging
- **JavaScript** — Frontend integration
- **SQLite** — Database

## Getting Started

```bash
git clone https://github.com/okekefrancis112/Chat-App-Api.git
cd Chat-App-Api
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
